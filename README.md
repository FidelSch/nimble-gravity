# Nimble Gravity

A simple project to demonstrate the use of React.

## Quick setup

1. Clone the repository:
```bash
   git clone https://github.com/FidelSch/nimble-gravity
```
2. Install dependencies:
```bash
   cd nimble-gravity
   npm install
   npm run dev
```

3. Edit .env file:

```bash
VITE_EMAIL_ADDR=some-email-addr
VITE_BASE_URL=base-url
```

4. Run:

```bash
npm run dev
```
